# ml_notebooks

A collection of Jupyter Notebook tutorials demonstrating artificial intelligence and machine learning concepts.

## Overview

This repository contains hands-on machine learning tutorials and implementations using Python. The notebooks cover various ML techniques and applications with practical examples.

## Contents

### Notebooks

- **machine_learning_tutorial.ipynb** - Titanic Survival Prediction
  - Comprehensive ML pipeline including data cleaning, exploration, and modeling
  - Demonstrates data preparation techniques
  - Features handling of missing values and categorical encoding
  - Uses the famous Titanic dataset for survival prediction

## Language

- **Jupyter Notebook** (100%)

## Getting Started

1. Clone the repository
2. Install required dependencies (pandas, numpy, seaborn, scikit-learn, etc.)
3. Open notebooks in Jupyter or Google Colab
4. Follow along with the tutorials and experiments

## Key Topics Covered

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Handling Missing Values
- Categorical Variable Encoding
- Feature Engineering
- Model Training & Evaluation

## Requirements

- Python 3.x
- pandas
- numpy
- seaborn
- scikit-learn
- matplotlib / plotly (for visualizations)
